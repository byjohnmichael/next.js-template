# Next.js Template

### 0.0.1

A Next.js template to jumpstart projects made By John Michael

## Features

- Next.js 16.0.1 — Framework
- React 19.2.0 — UI Library
- TypeScript 5 — Type Saftey
- Tailwind CSS 4- Styling
- Prettier 3.6.2 — Code Formatting
- ESLint 9 — Code Linting
- PostCSS — CSS Processing

## Getting Started Guide

1. Clone
    ```bash
    git clone https://github.com/byjohnmichael/next.js-template.git
    ```
2. Navigate
    ```bash
    cd next.js-template
    ```
3. Installation
    ```bash
    npm install
    ```
4. Develop
    ```bash
    npm run dev
    ```
5. Lint
    ```bash
    npm run lint
    ```
6. Format
    ```bash
    npm run format:check
    npm run format
    ```
7. Build
    ```bash
    npm run build
    ```
